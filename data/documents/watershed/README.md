# Watershed / conservation evidence (original captures)

**Collection:** `watershed/` · **Method:** captured web evidence + (future) primary grant docs

Source material on the **Maumee River watershed** burden around the BOSC corridor —
agricultural nutrient loading, conservation/BMP grants, and the receiving-water
baseline that the hydrology axis reasons over. Distinct from `regulatory/` (which
holds enforcement instruments like the 1996 CWA consent decree): this collection
is conservation/watershed *context*, not regulatory action.

Each binary keeps its **as-received name is recorded in the extraction's
`provenance` block** (source URL, capture date, sha256, content-verified date) —
see the mirrored `data/extracted/watershed/`.
