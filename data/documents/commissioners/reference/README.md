# Allen County government reference rosters (original records)

**Collection:** `commissioners/reference/` · immutable source evidence

Published county reference documents used to attribute parties named in the
Board of Commissioners record (see the
[minutes corpus](../minutes/README.md)). Raw bytes are never edited.

## Contents

| File | What |
|---|---|
| `Govt.-Structure-Org-Chart-1.pdf` | County government structure / org chart. |
| `Mayors-and-Zoning-Persons.pdf` | Roster of municipal mayors and zoning contacts. |
| `Township-Trustees-Fiscal-Officers.pdf` | Township trustees and fiscal officers. |

These are official county publications kept as received; treat any name/title as
of the document's own date.
