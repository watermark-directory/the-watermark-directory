## Documents

### `15-Core Service Requirements.pdf`

**Ohio CV/AV System Core Service Requirements**, prepared by AECOM for DriveOhio
(Ohio Department of Transportation connected/autonomous vehicle program).

Defines seven core services (CS-1 through CS-7) for the statewide Ohio CV/AV
infrastructure:

| Service | Description |
|---|---|
| CS-1 | Core Authorization |
| CS-2 | Data Distribution |
| CS-3 | Infrastructure Management |
| CS-4 | Location and Time |
| CS-5 | Map Management |
| CS-6 | Object Registration and Discovery |
| CS-7 | Privacy Protection |

CS-3-18 explicitly requires horizontal scaling at the data center layer. The CV/AV
edge network feeds data to central data centers for processing — this is a formal
state requirement document linking Ohio's roadway infrastructure to data center demand.

Filed as deliverable item 15 of the DriveOhio system design package. Prepared by AECOM.
