# Allen County Commissioners — meeting minutes & agendas corpus

Prebuilt index over the Board of Commissioners' published meeting record, scraped from
**commissioners.allencountyohio.com** (the CorpComm-built county site — see
[`../../legal/allen-county-web-vendor-audit.md`](../../legal/allen-county-web-vendor-audit.md)).
Raw PDFs are the immutable source under
[`data/documents/commissioners/minutes/raw/`](../../../documents/commissioners/minutes/raw/)
(634 files, Git-LFS-tracked). This directory holds the **derived** index bundle.

> Provenance: generated by the external `bistrozzi` tool (`source: allen_commissioners_minutes`,
> `generated_at` 2026-04-17). Imported into BOSC 2026-06-05. The OCR text layer is machine-extracted
> and **approximate** — verify any figure/date against the source PDF before quoting in a filing.
> The `local_path` column in `meetings.parquet`/`index.json` points at the generating machine's
> `/tmp/bistrozzi/...` staging path; the authoritative location is now `data/documents/commissioners/minutes/raw/<filename>`.

## Contents

| File | What |
|---|---|
| `meetings.parquet` | 634 meetings — `filename, meeting_date, kind, doc_type, url, local_path, ocr_pages` |
| `meeting-pages.parquet` | 2,283 OCR'd pages — `filename, meeting_date, doc_type, page, text, char_count` (the OCR text of record) |
| `meeting-refs.parquet` | 342 pre-extracted topic/subject hits — `filename, page, ref_type, ref_value, context` |
| `index.json` | Scrape provenance per meeting (`url`, `discovered_at_index`) |
| `manifest.json` | Build summary + ref-type totals |

## Shape

- **634 meetings** = 322 agendas (`A*`) + 308 minutes (`M*`); 2,283 OCR pages.
- `meeting_date` is parsed from the filename (`A011624` → 2024-01-16) — mostly 2024–2026; a few
  out-of-range values are parse artifacts, treat the filename as ground truth.
- **139 meetings carry ≥1 BOSC-relevant reference.** Pre-extracted ref counts:

  | ref_type | n | | ref_type | n |
  |---|--:|---|---|--:|
  | topic_easement | 108 | | topic_pump_station | 15 |
  | topic_annexation | 108 | | topic_forcemain | 12 |
  | subject_hume | 31 | | topic_cmar | 11 |
  | subject_bosc | 28 | | subject_bistrozzi | 8 |
  | topic_datacenter | 15 | | subject_cridersville_treatment | 3 |
  | | | | subject_general_dynamics | 2 |

  `subject_google` and `subject_general_dynamics` are near-silent (0 / 2) — consistent with the
  code-named-shell pattern: the record speaks of **BOSC / Bistrozzi / Hume**, not Google.

## Querying

```python
import pandas as pd
refs = pd.read_parquet("data/extracted/commissioners/minutes/meeting-refs.parquet")
# every BOSC mention with its surrounding text
refs[refs.ref_type == "subject_bosc"][["filename", "page", "context"]]
```

To read a hit in full: open `data/documents/commissioners/minutes/raw/<filename>` at the cited `page`.
