# Watershed / conservation extractions

Reviewed reads and research notes for the Maumee-watershed conservation evidence
captured under [`data/documents/watershed/`](../../documents/watershed/README.md),
plus secondary-literature findings the hydrology axis reasons over.

## Files

| File | What |
|---|---|
| `osu-lima-ag-runoff-treatment.capture.yaml` | Structured capture of the Allen SWCD OSU-Lima Agricultural Runoff Treatment System project page (URL, capture date, sha256, content-verified date). |
| `shedekar-edge-of-field.findings.md` | Findings note distilled from edge-of-field monitoring literature. |
| `shedekar-vinayak.bibliography.md` | Bibliography for the above. |

## Conventions

The `*.capture.yaml` provenance block (source URL, sha256, capture + content-verified
dates) is the chain-of-custody record for the captured web page. The `.md` findings
summarize **secondary** literature — cite the underlying papers, don't treat the
summary as primary data. Open follow-ups are tracked in
[`../legal/corpus-completeness-audit.md`](../legal/corpus-completeness-audit.md).
