# About this scaffold

This page is **MDX** — Markdown with components — rendered by
[`@astrojs/mdx`](https://docs.astro.build/en/guides/integrations-guide/mdx/).
Its presence proves the MDX toolchain is wired end to end.

Project BOSC is an agentic research platform that deconstructs public-records
source documents into reviewed structured data. This redesigned site (Epic #54)
reads the committed **content bundle** — the typed JSON feeds the Python data
tier emits — and renders four sections:

- **A. Home / About** — landing, disclaimer, methodology, the bigger picture
- **B. The BOSC site** — documents, records, timeline, exhibits, people & places
- **C. The Maumee watershed** — hydrology dashboards, watershed map, imagery
- **D. Wiki** — entity & concept pages with backlinks and a graph neighborhood

Narrative pages like this one migrate into a proper MDX content collection in
issue #69; the four sections land in #65–#68.
